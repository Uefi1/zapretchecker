# zapretchecker
bol-van/zapret Checker

![Image alt](https://github.com/Uefi1/zapretchecker/blob/main/2025-12-09_022322.jpg)

Привет данный софт нечто иное как простой чекер стратегий для https://github.com/bol-van/zapret
с использованием проблем возникнуть никаких не должно интерфейс так как интерфейс дружелюбный и интуитивно понятный:
1. Выбрали путь к исполняемуому файлу запрета
2. Выбрали бинарник (он нужен далеко не для каждой стратегии)
3. Остальное можете выбрать своё или оставить по умолчанию
4. Убедитесь пожалуйста что запрет закрыт и не используется перед нажатием на Start

Результаты будут находится в папке с софтом Results файл Found_время.txt будет содержать отчёт о всех стратегиях лучшие стратегии всегда будут сверху находится вверху например:
0/80 - было открыто 0 сайтов с текущей стратегией
10/80 - было открыто 10 сайтов с текущей стратегией

Так же обнаружите что в папке Results есть ещё одна папка BestConfig в ней будут содержаться лучший конфиг из проверенных софтом и сам ярлык Zapret два раза кликнув по которому вызовется zapret 
с уже лучшей стратегией (работает так .../winws.exe @Config.cfg)

Внимание так как СОФТ работает ассинхронно после нажатия на кнопку Stop пожалуйста немного подождите пока не получите сообщение что софт завершил работу

Ещё запускайте пожалуйста от имени администратора иначе запрет не будет запускаться и софт будет работать некорректно
